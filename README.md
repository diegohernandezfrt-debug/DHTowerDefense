# Medieval Defense – Tower Strategy Game

A wave-based medieval strategy game developed in Godot 4.2.2.

## Features
- 5 progressive levels
- Multiple warrior types
- Ranged and melee enemies
- Economy system
- Victory & Game Over screens
- Unlockable units

## Tech
- Godot 4.2.2
- GDScript
- Custom wave manager system
- EnemyManager & GameManager architecture

## How to Play
- Place warriors using glory points

- Complete all waves to win
